# Final Project (TEMP)

This is the final project for the TEMP course.

## How to install:

Create virtual environment:
```bash
python -m venv .venv
```

Enter virtual environment:
```cmd
.venv\Scripts\activate
# Or linux or macOS:
# source .venv/bin/activate
```

Install dependencies:
```bash
pip install -r requirements.txt
```

## How to run:
```cmd
.venv\Scripts\activate
python main.py
```
